## 🇺🇸 English Version

**Saku Tienda** is a high-end mobile shopping application (Android & iOS) designed for a seamless, modern e-commerce experience. Featuring a clean UI, smooth animations, and real-time inventory synchronization, it offers users a premium platform to discover and purchase products.

### 🌟 Key Features
- **Intuitive Shopping Experience**: Beautiful product listing with advanced search and category filtering.
- **Dynamic Shopping Cart**: Real-time cart management with instant price calculations.
- **Secure Checkout**: Streamlined payment flow with integrated shipping management.
- **Favorites & Personalization**: Users can save products and manage their profiles.
- **Real-time Notifications**: Order updates and promotional alerts via Push Notifications.

### 🛠️ Tech Stack
- **Framework**: [Expo](https://expo.dev/) (SDK 55) & [React Native](https://reactnative.dev/).
- **Navigation**: [Expo Router](https://docs.expo.dev/router/introduction/).
- **Backend**: [Firebase Firestore](https://firebase.google.com/docs/firestore) & [Authentication](https://firebase.google.com/docs/auth).
- **Styling**: [NativeWind](https://www.nativewind.dev/) (Tailwind CSS) & [Lucide Icons](https://lucide.dev/).
- **Animations**: [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/).

## 🇪🇸 Versión en Español

**Saku Tienda** es una aplicación móvil de compras de alta gama (Android e iOS) diseñada para una experiencia de e-commerce moderna y fluida. Con una interfaz limpia, animaciones suaves y sincronización de inventario en tiempo real, ofrece a los usuarios una plataforma premium para descubrir y adquirir productos.

### 🌟 Características Principales
- **Experiencia de Compra Intuitiva**: Listado de productos atractivo con búsqueda avanzada y filtros por categoría.
- **Carrito Dinámico**: Gestión de carrito en tiempo real con cálculos instantáneos de precios.
- **Checkout Seguro**: Flujo de pago optimizado con gestión integrada de envíos.
- **Favoritos y Personalización**: Los usuarios pueden guardar productos y administrar su perfil.
- **Notificaciones en Tiempo Real**: Actualizaciones de pedidos y alertas promocionales mediante Notificaciones Push.

### 🛠️ Stack Tecnológico
- **Framework**: [Expo](https://expo.dev/) (SDK 55) y [React Native](https://reactnative.dev/).
- **Navegación**: [Expo Router](https://docs.expo.dev/router/introduction/).
- **Backend**: [Firebase Firestore](https://firebase.google.com/docs/firestore) y [Authentication](https://firebase.google.com/docs/auth).
- **Estilos**: [NativeWind](https://www.nativewind.dev/) (Tailwind CSS) y [Lucide Icons](https://lucide.dev/).
- **Animaciones**: [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/).

## 🚀 Setup & Installation / Instalación

1. **Clone the repo / Clonar repo**:
   ```bash
   git clone https://github.com/trabbittbusiness-spec/saku-tienda.git
   cd saku-tienda
   ```

2. **Install dependencies / Instalar dependencias**:
   ```bash
   npm install
   ```

3. **Run the project / Iniciar proyecto**:
   ```bash
   npx expo start
   ```
